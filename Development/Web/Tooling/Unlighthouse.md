# Unlighthouse

## Qué es
Unlighthouse es una herramienta para **auditar el rendimiento web de todo un sitio**, no solo una página individual. Funciona como Lighthouse, pero con una **visión global**: analiza múltiples rutas y genera puntuaciones agregadas.

## Para qué sirve
- Detectar problemas de rendimiento a escala
- Identificar páginas lentas o problemáticas
- Tener una visión real del estado performance del sitio

## Uso básico
```bash
npx unlighthouse --site <tu-web>
```

Ejemplo:
```bash
npx unlighthouse --site https://example.com
```

## Qué analiza
- Performance
- Best Practices
- SEO
- Accesibilidad
- Métricas agregadas por página

## Cuándo usarlo
- Auditorías periódicas
- Antes de releases grandes
- Optimización de performance en sitios medianos/grandes
