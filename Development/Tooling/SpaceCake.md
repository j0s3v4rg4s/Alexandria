# SpaceCake

## Qué es
SpaceCake es una **interfaz gráfica de código abierto para Claude Code**. Disponible para Windows, macOS y Linux.

## Para qué sirve
- Usar Claude Code con una interfaz visual cómoda
- Monitorizar agentes ejecutándose en paralelo
- Ver coste, modelo y tokens en tiempo real
- Editor de código integrado

## Características
- ✓ Coste, modelo y tokens en tiempo real
- ✓ Monitoriza agentes en paralelo
- ✓ Editor de código integrado
- ✓ Multiplataforma: Windows, macOS y Linux
- ✓ Código abierto
